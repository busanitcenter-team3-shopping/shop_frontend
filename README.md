## 1. 프로젝트 소개

- **프로젝트 명** : 중고판매
- **기간** : 25.02.10 ~ 25.03.15
  {(프론트 : 25.02.10 ~ 25.02.20), (백엔드 : 25.02.21 ~ 25.03.15)}
- **인원** : 4 人
- **설계 목적** : 본 중고마켓 프로젝트는 소비자들이 쉽고 안전하게 중고 물품을 거래 할수 있는 플랫폼입니다. 직관적인 사용자 경험과 실시간 소통을 통해 사용자들이 편리하고 신뢰할 수 있는 거래 환경을 누릴 수 있도록 설계 되었습니다. 이 프로젝트는 젊은 팀원들의 창의적 아이디어와 열정을 바탕으로 신뢰와 안전을 최우선으로 하는 중고 거래 생태계를 구축하는 것을 목표로 합니다.

### 상품 CRUD 및 부가 기능

<details>
  <summary>상품추가</summary>

- 접속한 유저가 있으면 상품의 등록이 가능하고 입력값을 유효성 검사(front/back) 후 데이터 베이스에 저장
- 이미지는 public / images 폴더가 없으면 생성후 해당 폴더에 저장, 있으면 해당 이미지 해당 폴더에 저장 이후 상품에 추가
![상품추가](https://github.com/busanitcenter-team3-shopping/shop_backend/blob/dev/src/main/resources/static/gif/상품추가.gif)
</details>

<details>
  <summary>상품수정</summary>

- 로컬에 저장된 접속 유저의 id값과, DB에 저장된 상품의 등록자id를 서로 비교하여 일치하면 입력값을 새롭게 저장
- public / images 폴더에 있던 기존 사진을 제거 이후 새로운 이미지 추가
![상품수정](https://github.com/busanitcenter-team3-shopping/shop_backend/blob/dev/src/main/resources/static/gif/상품수정.gif)
</details>

<details>
  <summary>등록상품 확인</summary>

- DB의 상품 등록자 id가 접속된 유저와 일치하면 자신이 등록한 상품을 전체 확인 가능
![상품확인](https://github.com/busanitcenter-team3-shopping/shop_backend/blob/dev/src/main/resources/static/gif/상품확인.gif)
</details>

<details>
  <summary>상품삭제</summary>

- 로컬에 저장된 접속 유저(currentUser)와 DB내 저장된 상품의 등록자 id가 서로 일치하면 상품 삭제 시 뷰로 출력이되어 클릭 하면 DB에서 삭제 후 랜더링
![상품삭제](https://github.com/busanitcenter-team3-shopping/shop_backend/blob/dev/src/main/resources/static/gif/상품삭제.gif)
</details>

<details>
  <summary>상품 리스트</summary>

- DB에 저장된 상품을 `findAll`을 통해 전체 상품 출력
![상품리스트](https://github.com/busanitcenter-team3-shopping/shop_backend/blob/dev/src/main/resources/static/gif/상품.gif)
</details>

<details>
  <summary>상품 검색</summary>

- 검색창에 상품명을 입력하고 엔터를 누르면, 입력된 검색어와 일치하는 상품만을 필터링하여 결과 페이지로 이동
![상품검색](https://github.com/busanitcenter-team3-shopping/shop_backend/blob/dev/src/main/resources/static/gif/상품검색.gif)
</details>

<details>
  <summary>상품 찜하기</summary>

- 상품의 하트 이미지 클릭시 DB 테이블 favorite에 해당 상품의 id 및 현재 로그인된 user id값을 저장하여 색 변환
![상품찜](https://github.com/busanitcenter-team3-shopping/shop_backend/blob/dev/src/main/resources/static/gif/찜.gif)
</details>

<details>
  <summary>상품 찜취소</summary>

- 찜한 상품의 하트를 클릭시 favorite에 저장된 내용이 삭제 이후 다시 처음 하트이미지로 변환
![찜취소](https://github.com/busanitcenter-team3-shopping/shop_backend/blob/dev/src/main/resources/static/gif/찜취소.gif)
</details>

<details>
  <summary>메시지</summary>

- 웹소켓을 사용하여 실시간 메시지 전송 및 수신을 구현
- 클라이언트와 서버가 양방향 통신을 통해 채팅 데이터를 주고받음
- 메시지를 DB에 저장하여 채팅 내역을 유지하고, 새 메시지는 실시간으로 UI에 반영
![메시지](https://github.com/busanitcenter-team3-shopping/shop_backend/blob/dev/src/main/resources/static/gif/메시지.gif)
</details>

<details>
  <summary>상품 판매하기</summary>

- 채팅방 내 상품을 등록한 유저는 상품 판매 버튼이 나오고 버튼 클릭시 채팅방에 있는 다른 유저의 id값을 `purchase_history` 테이블에 저장하여 상품이 판매되도록 랜더링

<details>
  <summary>상품리뷰</summary>

## 8. 보완점

- 채팅 서비스내 채팅창 읽음 표시 재랜더링 기능 구현X
  - 같은 채팅방 내 상대방이 있을 경우 DB에 저장된 IS_READ가 false가 true로 변경이 되지만, 뷰에선 새로고침을 해야 true로 적용
